The sorcerer's apprentice

Descripción

The Sorcerer’s Apprentice es un Rogue like 2D en tiempo real que consta de un mapa con vista cenital que tendrá obstáculos y enemigos por cada nivel (con un total de 3 niveles).El mapa tendrá un tamaño de X tiles.El jugador podrá usar ataques cuerpo a cuerpo o distintos tipos de poderes que gastan una barra de maná con el objetivo de matar a todos los enemigos.Los poderes podrán combinarse entre sí para formar nuevos poderes. Al completar el nivel se conseguirá un nuevo poder (excepto en el último nivel). En cada nivel habrá objetos que soltarán dinero o fragmentos de maná al ser destruidos. El dinero lo usaremos para comprar power-ups al completar el nivel y estarán activos durante el siguiente.
