The sorcerer's apprentice

Desarrollado por Felipe Cuadra Plaza y Antonio Jesús Guerra Garduño

Publicado bajo MIT License

Descripción:

The Sorcerer’s Apprentice es un juego 2D en tiempo real con elementos de Rogue like que consta de un mapa con vista cenital que tendrá obstáculos y enemigos por cada nivel. El jugador podrá usar ataques cuerpo a cuerpo o distintos tipos de poderes que gastan una barra de maná con el objetivo de matar a todos los enemigos.Los poderes podrán combinarse entre sí para formar nuevos poderes. Al completar el nivel se conseguirá un nuevo poder. En cada nivel habrá objetos que soltarán dinero o fragmentos de maná al ser destruidos. El dinero lo usaremos para comprar power-ups al completar el nivel y estarán activos durante el siguiente.

Controles:

El movimiento del jugador se realizará con las teclas W(arriba), A(izquierda), S(abajo) y D(derecha).Para hacer el ataque físico se usa el click izquierdo del ratón.Para usar la magia seleccionada deberá apuntar con el ratón y apretar el click derecho.El menú de combinación de magias se abre con la tecla T. La magia actual se cambia con la tecla E.
